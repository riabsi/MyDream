# MyDream Project - لعبة غابة المغامرات الواقعية

## وصف المشروع (Project Description)
مشروع لعبة مغامرات طموح يهدف إلى تقديم تجربة واقعية في بيئة غابات جبلية. تعتمد اللعبة على فيزياء حقيقية لحركة الجسم وتفاعل الشخصية مع البيئة.

## الميزات الأساسية (Key Features)
* **بيئة واقعية:** غابات وأعشاب طبيعية بالكامل (ليست مجرد صور).
* **نظام تسلق حر:** إمكانية تسلق أي شجرة أو جبل داخل اللعبة.
* **فيزياء الجسم:** حركة مفاصل واقعية وتفاعل حركي مع التضاريس.
* **نظام الأسلحة والحقيبة:** قائمة أسلحة متنوعة (سيف، حبل تسلق، مسدس، كلاشينكوف).
* **تحكم الموبايل:** نظام Joystick للحركة مع أزرار مخصصة للقفز، الضرب، وفتح الحقيبة.

## نظام التحكم (Controls)
* **الجهة اليسرى:** زر دائري (Joystick) للمشي في جميع الاتجاهات.
* **الجهة اليمنى:** * زر القفز.
    * زر الهجوم/الاستخدام.
    * زر الحقيبة (Inventory) لاختيار الأسلحة.

## التكنولوجيا المستخدمة (Tech Stack)
* **المحرك:** Unity / Unreal Engine (قيد التطوير).
* **اللغة:** C# (C-Sharp).
